# DSA-Lab-Programs
Stack Implementation using Array
⚙️ Operations Used 
Push → Insert element into stack 
Pop → Remove top element
Peek → View top element
Display → Show all elements
⚠️ Conditions
Overflow → When stack is full
Underflow → When stack is empty 
⏱ Time Complexity
Push → O(1)
Pop → O(1)
Peek → O(1)
Linear queue implementation
⚙️ Operations
Enqueue → Insert element at rear
Dequeue → Remove element from front
Display → Show all elements
⚠️ Conditions
Overflow → When queue is full (rear == MAX - 1)
Underflow → When queue is empty (front > rear or front == -1)
⏱ Time Complexity
Enqueue → O(1)
Dequeue → O(1)

🔍 Binary Search (C)

Implementation of Binary Search algorithm in C to efficiently find an element in a sorted array.

🚀 Features

- Efficient searching (O(log n))
- Works on sorted arrays
- Simple and clean implementation

▶️ Run

gcc binary_search.c -o output
./output

📊 Output

Element found at index 2

🧠 Concept

Binary Search works by repeatedly dividing the search space into halves until the element is found.
